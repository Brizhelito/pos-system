# Guía de Atajos de Teclado - Sistema POS

El Sistema POS está diseñado para permitir una operación rápida y eficiente a través del teclado, reduciendo la necesidad de usar el mouse para la mayoría de las operaciones comunes. Esta guía describe todos los atajos de teclado disponibles en el sistema.

## Atajos Globales

Estos atajos funcionan en cualquier parte del sistema cuando estás autenticado:

| Atajo | Descripción |
|-------|-------------|
| `F1` | Mostrar ayuda contextual |
| `Ctrl + D` | Ir al Dashboard |
| `Ctrl + S` | Ir al módulo de Ventas |
| `Ctrl + P` | Ir al módulo de Productos |
| `Ctrl + C` | Ir al módulo de Clientes |
| `Ctrl + I` | Ir al módulo de Inventario (solo administradores) |
| `Ctrl + R` | Ir al módulo de Reportes (solo administradores) |
| `Ctrl + L` | Cerrar sesión |
| `Ctrl + ,` | Abrir configuración de usuario |
| `Alt + T` | Cambiar entre tema claro/oscuro |

## Módulo de Ventas

### Navegación y Acciones Rápidas

| Atajo | Descripción |
|-------|-------------|
| `F2` | Nueva venta |
| `F3` | Enfocar búsqueda de productos |
| `F4` | Abrir búsqueda de clientes |
| `F5` | Procesar pago |
| `F6` | Guardar venta como pendiente |
| `F7` | Recuperar venta pendiente |
| `F8` | Aplicar descuento a la venta |
| `F9` | Imprimir último recibo |
| `F12` | Alternar pantalla completa |
| `Esc` | Cancelar acción actual / Cerrar diálogo |
| `Ctrl + B` | Limpiar carrito |
| `Ctrl + H` | Ver historial de ventas del día |

### Gestión del Carrito

| Atajo | Descripción |
|-------|-------------|
| `Tab` | Navegar entre elementos del carrito |
| `+` | Incrementar cantidad del producto seleccionado |
| `-` | Disminuir cantidad del producto seleccionado |
| `Delete` | Eliminar producto seleccionado del carrito |
| `Ctrl + Up/Down` | Navegar entre productos en el carrito |
| `Alt + P` | Editar precio (si está permitido) |
| `Alt + D` | Aplicar descuento al producto seleccionado |
| `Ctrl + Enter` | Confirmar edición de cantidad |

### Búsqueda de Productos

| Atajo | Descripción |
|-------|-------------|
| `Enter` | Seleccionar producto resaltado |
| `Up/Down` | Navegar entre resultados de búsqueda |
| `Ctrl + F` | Búsqueda avanzada de productos |
| `Alt + C` | Filtrar por categoría |
| `Alt + S` | Ordenar resultados |
| `Esc` | Cerrar resultados de búsqueda |

### Proceso de Pago

| Atajo | Descripción |
|-------|-------------|
| `1-4` | Seleccionar método de pago (1: Efectivo, 2: Pago Móvil, 3: Transferencia, 4: Punto de Venta) |
| `Tab` | Navegar entre campos de pago |
| `Enter` | Confirmar pago |
| `Esc` | Cancelar proceso de pago |
| `M` | Habilitar pago mixto |
| `Alt + R` | Ver resumen de la venta |
| `Alt + N` | Añadir nota a la venta |

## Notas Importantes

- Los atajos pueden variar ligeramente dependiendo del navegador y sistema operativo
- En macOS, utiliza la tecla `Command (⌘)` en lugar de `Ctrl` para la mayoría de los atajos
- Algunos atajos pueden estar deshabilitados según el rol del usuario y los permisos
- Los atajos de función (F1-F12) pueden requerir la tecla `Fn` en algunos teclados de laptops

## Personalización de Atajos

Los administradores pueden personalizar algunos atajos de teclado a través del módulo de configuración del sistema:

1. Ve a Configuración > Sistema > Personalización
2. Selecciona la pestaña "Atajos de Teclado"
3. Modifica los atajos según sea necesario
4. Haz clic en "Guardar Cambios"

## Consejos para Principiantes

Si eres nuevo en el uso de atajos de teclado, comienza memorizando estos atajos esenciales:

- `F3`: Buscar productos durante una venta
- `F4`: Buscar/asignar cliente
- `F5`: Procesar pago
- `+` y `-`: Incrementar/reducir cantidades en el carrito
- `Esc`: Cancelar la acción actual o cerrar diálogos

A medida que te sientas más cómodo, puedes incorporar más atajos a tu flujo de trabajo para aumentar tu eficiencia.
